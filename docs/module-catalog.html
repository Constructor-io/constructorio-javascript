    Add item to index
    Source:
    • modules/catalog.js, line 51
    See:
    • https://docs.constructor.io/rest_api/items/add_an_item
    Parameters:
    Name Type Description
    parameters object Additional parameters for item details
    Name Type Attributes Description
    item_name string The name of the item, as it will appear in the results
    section string Your autosuggest and search results can have multiple sections like "Products" and "Search Suggestions". This indicates which section this item is for
    suggested_score number <optional>
    		 A number between 1 and 100 million that will influence the item's initial ranking relative to other item scores (the higher the score, the higher in the list of suggestions the item will appear)
    keywords Array.<string> <optional>
    		 An array of keywords for this item. Keywords are useful if you want a product name to appear when a user enters a search term that isn't in the product name itself
    url string <optional>
    		 A URL to directly send the user after selecting the item
    image_url string <optional>
    		 A URL that points to an image you'd like displayed next to some item (only applicable when URL is supplied)
    description string <optional>
    		 A description for some item (only applicable when URL is supplied)
    id string <optional>
    		 An arbitrary ID you would like associated with this item. You can use this field to store your own ID's of the items to more easily access them in other API calls
    facets object <optional>
    		 Key/value pairs that can be associated with an item and used to filter them during a search. You can associate multiple values with the same key, by making values a list. Facets can be used as filters in search, autosuggest, and browse requests
    metadata object <optional>
    		 You can associate schema-less data with items by passing in an object of keys and values. To configure search and display of this data reach out to support@constructor.io.
    group_ids Array.<string> <optional>
    		 You can associate each item with one or more groups (i.e. categories). To set up a group hierarchy please contact support@constructor.io. group_ids can be used as filters in search, autosuggest, and browse requests
    variations Array.<object> <optional>
    		 List of this item's variations
    Returns:
    Type:
